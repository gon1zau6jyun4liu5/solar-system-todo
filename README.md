## 🌌 v0.4.6 핵심 개념 - 동적 태양계 구조

### 🌟 **태양 (Sun) - AI 자동 그룹핑**
- 입력된 태스크를 AI가 자동으로 분석하여 **그룹 생성**
- 각 그룹이 하나의 **태양**이 되어 독립적인 태양계 형성
- 그룹핑은 고정이 아닌 **AI가 동적으로 생성**
- **2개 이상의 그룹**이 생기면 **다중 태양계** 시스템 구현

### 🪐 **행성 (Planet) - 그룹 내 태스크**
- 각 태양 그룹 내의 개별 **태스크**가 행성이 됨
- 행성에는 **태스크의 키워드**가 표시됨
- **태스크 삭제 시** 해당 행성이 사라짐
- 행성 클릭 시 **상세정보 카드** 표시

### 🌙 **위성 (Satellite) - 서브 태스크**
- 각 행성(태스크)의 **서브 태스크**가 위성이 됨
- 위성에도 **서브 태스크 키워드** 표시
- **서브 태스크 삭제 시** 해당 위성이 사라짐
- 위성 클릭 시 **서브 태스크 상세정보** 표시

### ☄️ **소행성 (Asteroid) - AI 액션 제안**
- AI가 **임의의 타이밍**에 **임의의 태스크**에 대해 **액션 제안**
- 소행성은 **해당 태스크(행성) 방향으로 이동**
- **기간 내 액션 처리** 시 소행성 사라짐
- **기간 초과** 시 해당 태스크(행성)에 **충돌** 효과

## ✨ v0.4.6의 새로운 기능

### 🤖 **AI 기반 동적 그룹핑**
- **자동 태스크 분석** - 입력된 태스크의 내용, 맥락, 관련성 분석
- **스마트 그룹 생성** - 유사한 태스크들을 자동으로 그룹화
- **동적 태양계 생성** - 그룹 수에 따라 1개 이상의 태양계 생성
- **실시간 재그룹핑** - 새 태스크 추가 시 기존 그룹 재검토

### 🌌 **다중 태양계 시스템**
- **복수 태양계 지원** - 2개 이상의 그룹 발생 시 여러 태양계 동시 표시
- **태양계 간 거리 조정** - 각 태양계가 겹치지 않도록 적절한 간격 유지
- **개별 태양계 제어** - 각 태양계별로 독립적인 상호작용 가능
- **전체/개별 뷰 전환** - 모든 태양계 또는 특정 태양계 집중 보기

### 🎯 **인터랙티브 천체 시스템**
- **태양 클릭** - 해당 그룹의 모든 태스크 목록 표시
- **행성 클릭** - 개별 태스크 상세정보 및 편집 카드
- **위성 클릭** - 서브 태스크 관리 및 상태 변경
- **소행성 추적** - 이동하는 소행성의 목적지와 남은 시간 표시

### ☄️ **AI 액션 제안 시스템**
- **스마트 제안 생성** - 태스크 상태와 우선순위를 분석하여 액션 제안
- **타이밍 최적화** - 사용자의 활동 패턴을 학습하여 최적의 제안 시점 결정

## 🌟 핵심 시스템 구조

### 🏗️ **계층 구조**
```
태양계 레벨 (Solar System Level)
├── 태양 (Sun) - AI 그룹
│   ├── 행성 1 (Planet) - 태스크 1
│   │   ├── 위성 1-1 (Satellite) - 서브 태스크 1
│   │   └── 위성 1-2 (Satellite) - 서브 태스크 2
│   ├── 행성 2 (Planet) - 태스크 2
│   │   └── 위성 2-1 (Satellite) - 서브 태스크 1
│   └── 소행성 (Asteroid) - AI 액션 제안
│       └── → 행성 1로 이동 중
└── 다른 태양계 (다른 AI 그룹)
```

### 🔄 **동적 시스템 흐름**
1. **태스크 입력** → AI 분석 시작
2. **그룹 결정** → 기존 그룹 또는 새 그룹 생성
3. **태양계 구성** → 태양-행성-위성 구조 생성
4. **소행성 생성** → AI가 필요 시 액션 제안
5. **상호작용** → 사용자 클릭으로 관리
6. **동적 업데이트** → 실시간 구조 변경

## 🔧 기술 구현

### 🤖 **AI 그룹핑 엔진**
```javascript
// AI 태스크 분석 및 그룹핑 시스템
const analyzeAndGroup = async (tasks) => {
  const analysis = await AIEngine.analyzeTasks(tasks);
  const groups = await AIEngine.createGroups(analysis);
  return groups.map(group => ({
    id: generateId(),
    name: group.name,
    tasks: group.tasks,
    theme: group.theme,
    priority: group.priority
  }));
};
```

### 🌌 **동적 태양계 생성**
```javascript
// 다중 태양계 렌더링 시스템
const renderSolarSystems = (groups) => {
  return groups.map((group, index) => (
    <SolarSystem
      key={group.id}
      position={calculateSystemPosition(index, groups.length)}
      sun={group}
      planets={group.tasks}
      satellites={group.tasks.flatMap(task => task.subtasks)}
      asteroids={getAsteroidsForGroup(group.id)}
    />
  ));
};
```

### ☄️ **소행성 이동 시스템**
```javascript
// 소행성 물리 엔진
const updateAsteroidPosition = (asteroid, targetPlanet, deltaTime) => {
  const direction = targetPlanet.position.clone()
    .sub(asteroid.position)
    .normalize();
  
  asteroid.position.add(
    direction.multiplyScalar(asteroid.speed * deltaTime)
  );
  
  // 충돌 감지
  if (asteroid.position.distanceTo(targetPlanet.position) < COLLISION_THRESHOLD) {
    triggerCollisionEffect(asteroid, targetPlanet);
  }
};
```
